# MJU Internet Programming
## Code-Share-API

### Install & Run
```bash
$ git clone https://github.com/TimeTREE98/code-share-be
$ cd code-share-be
$ docker-compose up -d
$ nvm use # or nvm install
$ pnpm install
$ pnpm dev
```
